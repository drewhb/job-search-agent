# Master Resume

Put your current canonical resume here. This is the baseline for all tailoring — it is **never edited in place** when tailoring for a specific role.

**Naming convention:** `YourName_Resume_v[N].docx` (e.g., `Alex_Chen_Resume_v3.docx`)

The `tailor-resume` skill reads from this folder and saves tailored output to `applications/[Company]_[Role]/` — it never writes back here.

When you make significant updates to your actual resume (new job, big project, substantial rewrites), save the new version here with an incremented version number and update the filename reference in `AGENT.md` (or `CLAUDE.md`).
